DynamicSPARQL dynamically scaffolds SPARQL queries.
Please visit project wiki pages https://github.com/Efimster/DynamicSPARQL/wiki  
Available through NuGet https://www.nuget.org/packages/DynamicSPARQL/

Since v1.0.2.1, Querying function has been changed as the result of decoupling with dotNetRDF.

DynamicSPARQL dotNetRDF connector binds dotNetRDF with DynamicSPARQL. 
Available through NuGet https://www.nuget.org/packages/DynamicSPARQL.dotNetRDF/

DynamicSPARQL BrightstarDB connector binds BrightstarDB with DynamicSPARQL. 
Available through NuGet https://www.nuget.org/packages/DynamicSPARQL.BrightstarDB/
